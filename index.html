🔬 Under The Hood

How RAG Technology Powers Our AI

Retrieval-Augmented Generation (RAG) is the architecture behind all our products. Here's how a user question becomes an accurate AI response in milliseconds.

User Query
Student or user asks a natural language question
embed
Vector Embedding
Query converted to semantic vector via Azure OpenAI
search
Vector Store
MongoDB Atlas finds the most semantically similar chunks
retrieve
LLM Generation
Gemini / GPT generates answer grounded in retrieved context
respond
Accurate Answer
User receives a precise, sourced, real-time response
